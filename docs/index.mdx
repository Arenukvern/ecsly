# ecsly

`ecsly` is an experimental Entity Component System package family for Dart and
Flutter.

The public repo currently contains four publishable package lanes:

- `ecsly`: the pure Dart ECS runtime.
- `ecsly_app`: app-layer actions, drafts, invalidation, and identity helpers.
- `ecsly_codegen`: annotations and builder support for typed-column factories.
- `ecsly_flutter`: Flutter bindings for worlds, selectors, actions, and loops.

The present claim is intentionally modest: this is an early prerelease R&D
package family with a stabilizing core. The North Star is a practical Dart game
engine stack over time, but only the packages present in this public repo are
canonical here.

Start with the repo
[architecture](https://github.com/Arenukvern/ecsly/blob/main/ARCHITECTURE.md)
for ownership, the [package map](/packages) for package lanes, and the package
READMEs for API details.
