# Contributors

This project uses [all-contributors](https://allcontributors.org/) for visible
contributor credit.

The canonical config is `.all-contributorsrc`. The rendered roster lives in the
README so it is visible on GitHub and package landing pages.

## Add A Contributor

Use the bot or CLI from a pull request:

```sh
npx all-contributors-cli add <github-login> code,doc
npx all-contributors-cli generate
```

Commit both `.all-contributorsrc` and `README.md`.

Common contribution types:

- `code`
- `doc`
- `bug`
- `userTesting`
- `infra`
- `security`
- `research`
- `maintenance`
- `tutorial`

Choose contribution types that describe what actually happened.
