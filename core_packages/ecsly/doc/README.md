runtime publication contracts unless they are explicitly migrated here.

## Publishing hygiene

- Keep runtime-facing docs concise and stable.
- Keep design/usage pages aligned with package behavior.
- Move drifted notes into their owning documents and keep links explicit.
- Add a short changelog note for any API-level behavior change.
